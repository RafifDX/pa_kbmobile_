# pa_kbmobile
Banana classifier
Aplikasi Banana Classifier adalah sistem yang dirancang untuk secara otomatis mengidentifikasi dan mengkategorikan gambar pisang berdasarkan karakteristik visualnya, seperti tingkat kematangan (mentah, matang, busuk). Inti dari kemampuan klasifikasi ini adalah model Convolutional Neural Network (CNN).

Cara menggunakan aplikasi:
- Ketuk tombol "Scan Pisang" di bagian bawah halaman utama.
- Pilih foto dari galeri atau ambil foto dengan kamera.
- Aplikasi akan menganalisis tingkat kematangan pisang.
- Hasil klasifikasi akan ditampilkan beserta rekomendasinya.

Cara menjalankan server:
Ketik python manage.py runserver di folder api. Lalu server akan berjalan.
